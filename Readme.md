# ViaSegura – Infraestrutura DevOps

Este repositório contém toda a **infraestrutura DevOps** do projeto ViaSegura, incluindo:

- Stack de monitoramento com **Zabbix** e **Grafana** (via Docker Compose)
- Scripts de automação (start/stop/update)
- Estrutura pronta para **CI/CD** e integração com os repositórios de Front-end e Back-end
- Base inicial para uso futuro de **Terraform** na criação de infraestrutura AWS
